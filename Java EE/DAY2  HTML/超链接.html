超链接标签 一、链接资源 href 用于指定链接的资源 target 设置打开资源的目标 _Blank 在当前窗口打开新资源 _self 在新窗口打开新资源 file （文件协议）用于搜索本地及其的资源文件 格式：file:\\\f:\美女\1.jpg mailTo 邮件协议 thunder 迅雷协议 例： 超链接 百度 发送邮件 123456@qq.com 迅雷下载链接 一个人的武林（高清）.avi ---------------------------------------------- a标签的原理： 判断href的属性值 1）以http开头 -> 启动http解析器，在本机查看host文件有无对应主机-> 没有 -> 去浏览器寻找对应的DNNS服务器 -> 有 -> 请求直接发给对应主机 （当做本地资源） 2）没有以任何协议开头 -> 就会以启动file协议，去解析该路径 3）以其他的协议开头 -> 浏览器去到本机的注册表中 -> 查找处理这种程序的应用程序 二、锚点定为 -- 中间是内容 -- 回到顶部 其中的top为 自定义锚点名字
